Vyntrix is a lightweight backend API for tracking and managing user subscriptions.
Built with **Node.js** and **Express**, it provides clean REST endpoints for creating, updating, listing, and deleting subscriptions.

## Tech Stack

* **Node.js**
* **Express**
* **MongoDB Atlas**
* **Mongoose**
* **Arcjet**

## Quick Setup

```bash
git clone "https://github.com/Iniyaa21/Vyntrix.git"
cd Vyntrix
npm install
```

Create an environment file `.env.development.local` and copy and paste the contents of `.env.example`. Update the values according to your environment.

## Run the Server

```bash
npm run dev
```
